# 🌸 えみラボ体操教室アプリ Ver.1

## 📌 プロジェクト概要

**えみラボ体操教室アプリ**は、先生の事務負担を減らし、生徒とのつながりを深めることを目的として開発した体操教室専用アプリです。

先生はGoogleカレンダーやGoogleスプレッドシートを利用して簡単に情報を更新でき、生徒はスマートフォンからいつでもレッスン動画やスケジュール、お知らせを確認できます。
